A watch face for a Pebble watcht inspired by the Stardew Valley clock.
You can download the watchface here: https://apps.repebble.com/954dddca9a9c44f0ab1c3ff8
The watch face includes:
- The time
- The date
- The number of steps you have taken (the counter below)
- A dial that shows whether it is day or night
- The weather (I could not test this on a real watch)
- The current season
I hope you enjoy using it!
